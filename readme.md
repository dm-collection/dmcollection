# DM Collection

⚠️ Under Construction ⚠️

### Running locally

Check [docker-compose.yaml](docker-compose.yaml). The server needs card data in
a `data/card-data.json` file. The schema is at [schema.json](server/schema.json).

Card images also need to be provided.

### Project structure

`/ui`: contains svelte-based web UI project
`/server`: contains backend server based on Spring Boot
